# Meta Plus

**TYPO3 CMS extension to manipulate:**
- absolute page title
- page title suffix
- page title prefix
- OpenGraph image
- favicon

## System requirements
- TYPO3 8.7 LTS – 9.*

## Installation
-  Install from TER: **metaplus** or Composer: **t3brightside/metaplus**
-  Include static template

## Usage

Edit page properties, new fields appear in “Metadata” tab in "Meta Plus" section.
